# DaliyCostBook

一个简单的日常记账的H5应用

HTML界面基本全部使用weui来完成的

第一版是使用asp.net的风格完成的网络交互，后来有些地方觉得不爽，就把所有的前后端交互都改成ajax了

使用CYQ.DATA完成的数据库交互操作，建库脚本dbo.sql，这个脚本是sqlserver生成的
